# ZigZag 3D

## Descripción

**ZigZag 3D** es un videojuego desarrollado en **Unity utilizando C#**, inspirado en los juegos arcade de reflejos rápidos.

El jugador controla un personaje que se desplaza automáticamente hacia adelante sobre un camino estrecho formado por **muros generados de manera procedural**. Utilizando la **tecla espacio**, el jugador cambia la dirección del movimiento (izquierda o derecha) para mantenerse en el camino.

El objetivo es **recolectar la mayor cantidad posible de cristales** mientras evita caer al vacío.

A medida que el jugador avanza, el desafío aumenta debido a la velocidad y precisión necesarias para mantenerse en el camino.

---

## Mecánicas principales

- Movimiento automático del jugador.
- Cambio de dirección con la **tecla espacio**.
- **Generación procedural del camino**.
- Sistema de **recolección de cristales**.
- Sistema de **puntuación basado en cristales recolectados**.
- Fin del juego cuando el jugador cae.
